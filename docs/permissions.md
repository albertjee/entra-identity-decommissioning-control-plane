# Permissions

## Microsoft Graph Delegated Scopes

- User.ReadWrite.All
- Directory.ReadWrite.All
- Organization.Read.All
- RoleManagement.Read.Directory
- Application.Read.All
- AppRoleAssignment.ReadWrite.All
- DelegatedPermissionGrant.Read.All

## Exchange Online

The operator must have sufficient permissions to:

- Read mailbox properties
- Convert mailbox to shared
- Configure auto-reply
- Enable Litigation Hold
- Read hold and archive indicators

## Operational Note

Some compliance and retention signals may require additional Purview or Exchange administrative roles depending on tenant configuration.
